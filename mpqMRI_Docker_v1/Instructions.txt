Before you begin make sure you have the docker desktop app running on your computer
1. Navigate to the docker folder with windows powershell
2. run the docker image using the command :
 docker build -t qcet .
3. Make a folder with the data : The folder should contain six files named in this way:
M0w_mag.nii
T1w_mag.nii
M0w_pha.nii
T1w_pha.nii
B1_1.nii
B1_2.nii
Note: both .nii and .nii.gz files can be used as input files
4. Run the docker using this command:
docker run -it --name mpqMRI -v "path_to_your_input_folder:/app/input" qcet python3 /app/script_qCET.py --input-folder /app/input
Note: there is another script that allows the user to change the parameters used. To run it instead of the regular script use this command:
 docker run -it --name mpqMRI -v "path_to_your_input_folder:/app/input" qcet python3 /app/script_user_interaction_test.py --input-folder /app/input
